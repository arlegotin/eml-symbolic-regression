# Phase 43 Plan: Paper Decision Memo and Completeness Boundary

**Status:** Ready
**Requirements:** PAP-01, PAP-02, PAP-03, PAP-04, PAP-05

## Goal

Users can make a publication decision from centered-family evidence with safe claims and explicit boundaries.

## Tasks

1. Add a paper-decision package generator that reads optional aggregate JSON evidence and writes Markdown/JSON outputs.
2. Encode the publish-vs-wait decision logic so missing centered evidence remains explicit.
3. Write safe claim, unsafe claim, figure/table inventory, and completeness-boundary artifacts.
4. Add a CLI command for reproducible memo generation.
5. Add tests for the generator and CLI command.

## Success Criteria

1. Decision memo chooses a paper posture from available evidence and states when empirical centered-family evidence is missing.
2. Safe claims cover centering, normalized local Jacobian, curvature control, shifted singularity, and subtraction-limit behavior.
3. Unsafe claims cover unproved completeness, `ZEML_s` terminal-0 limitations, universal recovery, and pocket-calculator replacement claims.
4. Figure/table inventory names exact recovery versus depth plus anomaly, verifier, repair/refit, and overhead tables.
5. Completeness boundary output is inspectable and labels constructive completeness evidence as incomplete unless supplied.
